# Schooler

## What is Schooler

Schooler is an information platform for schools. Allowing students easier access to the resources, including timetable and academic calendar.

## Tech Stack
### Frontend
- Next.js
- Auth.js
- ShadcnUI
### Backend
- FastAPI
- sqlmodel / SQLAlchemy
- fastapi-nextauth-jwt

## How to run
### Frontend
Deploy the frontend on platforms like Vercel, or build using `pnpm build`

### Backend
Please see the sample docker-compose.yml
